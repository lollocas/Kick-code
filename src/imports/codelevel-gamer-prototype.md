Quiero que diseñes un prototipo visual de una página web educativa para practicar programación. No es necesario que funcione ni que tenga lógica real, solo necesito la parte visual (interfaz).



Estilo de diseño



El diseño debe tener un ambiente gamer moderno.



Características del estilo:

• Colores principales: morado oscuro, negro y tonos neón morados.

• Fondo negro o muy oscuro.

• Botones con brillo o efecto neón morado.

• Tipografía moderna estilo tecnología/gamer.

• Paneles o tarjetas con bordes redondeados.

• Diseño limpio, moderno y atractivo para estudiantes.

• Puede parecer una plataforma tipo editor de código gamer.



La página debe contener 6 secciones o pantallas que representen las siguientes historias de usuario.



⸻



1. Ejercicios de sintaxis (HU-01)



Crear una sección donde el estudiante pueda practicar ejercicios de sintaxis de programación.



Elementos visuales:

• Título: “Ejercicios de Sintaxis”

• Un cuadro con el enunciado del ejercicio

• Un editor de código oscuro (estilo Visual Studio Code)

• Un botón morado que diga “Verificar sintaxis”



Ejemplo de ejercicio mostrado en pantalla:



Corrige el error de sintaxis:



for i in range(5)

    print(i)





⸻



2. Detector automático de errores (HU-02)



Mostrar un editor de código donde el sistema detecta errores de sintaxis.



Elementos visuales:

• Editor de código oscuro

• Una línea marcada en rojo indicando error

• Un mensaje debajo que diga:



“Error de sintaxis detectado en la línea 1: falta ‘:’ “



Debe verse como si el sistema estuviera analizando el código automáticamente.



⸻



3. Retroalimentación inmediata (HU-03)



Crear una sección donde el estudiante pueda ejecutar código y recibir retroalimentación inmediata.



Elementos visuales:

• Editor de código

• Botón morado que diga “Ejecutar código”

• Panel de resultados



Ejemplo de resultado visual:

• ✔ Código correcto (en verde)

• Mensaje: “Tu código funciona correctamente”



⸻



4. Selección de lenguaje de programación (HU-04)



Crear una sección para que el estudiante pueda elegir el lenguaje de programación.



Elementos visuales:

• Título: “Selecciona el lenguaje”

• Tarjetas o botones con lenguajes:



Python

Java

C++

JavaScript



Cada tarjeta puede tener iconos o estilo gamer.



⸻



5. Ejercicios por tipo de proyecto (HU-05)



Crear una sección donde el estudiante pueda elegir ejercicios basados en proyectos.



Diseñar tarjetas o botones con ejemplos como:

• Calculadora

• Sistema de Inventario

• Juego simple

• Gestión de estudiantes



Cada tarjeta debe tener iconos y efecto visual moderno.



⸻



6. Sistema de dificultad adaptativa (HU-06)



Crear una sección que muestre el nivel del estudiante y su progreso.



Elementos visuales:

• Texto: Nivel actual: Intermedio

• Barra de progreso morada

• Indicador de ejercicios completados

• Mensaje que diga:



“Has mejorado tu rendimiento. Se desbloquea el nivel avanzado.”



⸻



Diseño general



La página debe incluir:

• Barra superior (navbar) con el nombre de la plataforma:

CodeLevel Gamer

• Un menú con opciones como:

• Inicio

• Ejercicios

• Lenguajes

• Proyectos

• Progreso

• Todo el diseño debe verse como una plataforma gamer para aprender programación.



El resultado debe ser una interfaz visual clara y moderna, aunque no funcione realmente.
